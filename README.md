# commit
Simple, survey-based commit CLI 

```
$ go get github.com/jmaicher/commit
$ commit
? Tag  [Use arrows to move, type to filter]
> feat
  fix
  style
  refactor
  test
  docs
  clean
? Subject Add new feature
? Issue JIRA-9041
? Scope
[master f259d20] JIRA-9041 feat: Add new feature
 1 file changed, 87 insertions(+)
```
